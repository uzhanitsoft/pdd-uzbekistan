# 🚦 Yo'l harakati qoidalari — PDD O'zbekiston

Premium React web-ilova — O'zbekiston PDD imtihoniga tayyorlanish.

## ✨ Xususiyatlar

- 📋 **800+ real savol** — O'zbekiston PDD testlari
- 🖼️ **700+ HD rasm** — yo'l vaziyatlari
- 📱 **Mobile-first dizayn** — 430px max-width
- 🎯 **30 bilet** — har biri 20 savoldan
- ⏱️ **20 soniya timer** — har bir savol uchun
- 🔄 **UZ / RU** — til almashtirish
- 💾 **localStorage** — progress saqlanadi
- 🎉 **Confetti** — imtihon topshirganda
- 🃏 **Flip kartochkalar** — mashq rejimi
- ✨ **Framer Motion** — premium animatsiyalar

## 🛠 Texnologiyalar

- React 18 + Vite 6
- Tailwind CSS 3
- Framer Motion
- canvas-confetti
- Express (production)
